# MediIntel Architecture Notes

## Frontend

- App Router structure with protected application routes
- Demo-mode auth context for role-aware navigation
- Tailwind and shadcn/ui component foundation
- Framer Motion enabled for polished page transitions and login presentation

## Backend

- FastAPI application entrypoint
- Versioned API router scaffold
- JWT utility placeholders for future auth workflows
- SQLAlchemy and Alembic foundation for PostgreSQL

## Data Layer

- PostgreSQL connection configured through environment variables
- No entities or tables defined yet
- Alembic version folder created for future schema migrations
